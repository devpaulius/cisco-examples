# Network Exam Prep Repository

This repository is a ready-to-edit starter pack for:
- Cisco Packet Tracer labs
- Cisco IOS configuration snippets
- Linux command notes
- Windows VM / RDP setup notes
- Nmap scan notes
- Report templates
- Small helper scripts

## What is included
- `cisco-configs/` -> router and switch configs grouped by topic
- `linux/` -> Linux basics, networking, permissions, services
- `vm/` -> Windows VM, RDP, Zenmap, Wireshark notes
- `notes/` -> short study notes
- `report/` -> report templates and scan result templates
- `scripts/` -> local helper scripts
- `packet-tracer/placeholders/` -> text placeholders for `.pkt` files you will create yourself

## Suggested use
1. Create your own `.pkt` Packet Tracer files in `packet-tracer/`
2. Copy or adapt configs from `cisco-configs/`
3. Add your own notes to `notes/`
4. Keep screenshots in `report/screenshots/`
5. Use `scripts/search.sh` to find commands quickly

## Important
This pack does not include Packet Tracer `.pkt` files. It includes the configs and notes so you can build the topology yourself.
